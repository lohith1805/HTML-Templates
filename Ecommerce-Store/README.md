# Woodora Home - Ecommerce Store

Final-stage ecommerce template update based on the original project zip.

## Updated Features
- Furniture and home decor ecommerce homepage
- Offer bar, search-style navbar, cart and category navigation
- Smooth same-page navbar scrolling
- Fly-in animation from bottom while scrolling
- Rich homepage sections: hero, categories, deals, products, trust cards, reviews and footer
- Product listing page with search and category filters
- Cart page with quantity update, remove, empty cart and checkout button
- Checkout page with contact, delivery address, shipping, payment, billing and order summary
- Checkout form validation using HTML5 + JavaScript

## Pages
- `index.html`
- `products.html`
- `cart.html`
- `checkout.html`

## Note
This is a frontend-only template. Cart data is stored in browser localStorage for demo purposes.
