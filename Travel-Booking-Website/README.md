# SkyTrail Travels - Travel Booking Website Template

This template is built for the Travel Booking Website assignment tasks up to Day 2.

## Pages
- `index.html` - homepage with navbar, hero section, horizontally scrollable destination cards, feature cards, and contact form.
- `booking.html` - booking form page with JavaScript validation.

## Technologies
- HTML
- CSS
- JavaScript
- Bootstrap

## Implemented Day 1 Tasks
- Homepage design
- Hero section
- Destination cards
- Navigation

## Implemented Day 2 Tasks
- Booking form
- JavaScript validation
- Responsive layout

## How to Run
Open `index.html` in any browser, or use the VS Code Live Server extension.
