## 🛠️ Customization

### Update Personal Info
- Change name/title in the navbar
- Update hero section headline
- Replace profile image URL
- Update email and social links

### Add/Remove Skills
Edit the Skills Grid section and add Font Awesome icons:
```html
<div class="skill-card" title="Your Skill">
    <i class="fab fa-icon-name"></i>
    <span>SKILL NAME</span>
</div>
```

### Update Projects
Modify the project cards with your own projects and images.
