# Nestora Realty - Real Estate Website

Day 1 + Day 2 frontend template based on the provided real estate reference layout.

## Included
- Home page with hero section
- Reference-style search filter section
- Property cards
- Property detail page
- Embedded Google Map on the property detail page
- Responsive Bootstrap layout

## Technologies
- HTML
- CSS
- JavaScript
- Bootstrap

Open `index.html` in a browser to start.
