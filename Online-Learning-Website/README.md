# SkillNest Online Learning Website

Frontend online learning template created for Day 1 and Day 2 assignment tasks.

## Pages
- `index.html` - homepage with navigation, hero section, course categories and course cards

## Features
- Blue responsive navbar
- Navbar hides after scrolling down and appears when the cursor moves to the top of the page
- Homepage hero section
- Course category cards
- Creative course cards inspired by modern glass card UI
- Course filtering with JavaScript
- Responsive Bootstrap layout

## Technologies
- HTML
- CSS
- JavaScript
- Bootstrap

## How to run
Open `index.html` in a browser or use VS Code Live Server.
