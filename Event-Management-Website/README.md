# Eventora - Event Management Website

Updated final-stage event management template.

## Included
- Home page with event filters and audience filters
- Separate event detail pages inside `event-pages/`
- Gallery page connected from navbar
- Payment page redirected from Confirm Ticket / Book Ticket buttons
- Rich About section and footer contact information
- Smooth navbar scrolling on home page
- Fly-in animation from bottom on scroll
- Responsive Bootstrap-based layout

Open `index.html` with Live Server to test.
