# CareBridge Hospital Website - Day 1 + Day 2

Frontend hospital management template built with HTML, CSS, JavaScript and Bootstrap.

## Pages
- `index.html` - hospital homepage with hero, animated stats, about section, services and doctors
- `appointment.html` - appointment booking form with JavaScript validation

## Updates in this version
- Blue/white hospital color scheme
- Medilab-inspired navbar and font style
- Full-screen doctor background hero image
- About section with image only, no video/play button
- Fly-in reveal animation for page sections
- Navbar hides while scrolling down and returns while scrolling up
- Mobile responsive layout
