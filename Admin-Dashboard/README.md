# Vertex Admin Dashboard

A professional, responsive admin dashboard template with modern design, refined typography, and comprehensive UI components.

## Features

- **Responsive Sidebar & Navigation** - Collapsible sidebar with smooth transitions
- **Dashboard Cards** - Key metrics with icons and status indicators
- **Advanced Charts** - Interactive bar and line charts using Canvas
- **Data Tables** - Sortable tables with status badges and user indicators
- **Professional Typography** - System fonts with optimized sizing and spacing
- **User Symbols** - Icon-based user representations (no profile pictures)
- **Refined Styling** - Clean, modern color scheme with smooth animations
- **Mobile Optimized** - Fully responsive design for all devices

## Pages

- `index.html` - Main dashboard with metrics, charts, and quick actions
- `charts.html` - Dedicated analytics page with performance charts
- `tables.html` - Comprehensive data management and record viewing

## Responsive Breakpoints

- **Desktop:** Full layout with sidebar
- **Tablet (< 992px):** Collapsible sidebar
- **Mobile (< 576px):** Optimized for small screens

## Customization

Edit `css/style.css` to modify:
- Color variables in `:root`
- Typography and spacing
- Component styling

Edit `js/script.js` to:
- Modify chart data
- Change chart colors
- Update interactive behaviors
