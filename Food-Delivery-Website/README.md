# Food Delivery Website - Homepage Template

This is a frontend-only food delivery homepage template based on the assignment scope.

## Scope Completed

- Homepage
- Food cards
- Navigation
- Bootstrap responsive layout
- Simple food search and category filter using JavaScript

## Theme

The template is designed around one restaurant: **SpiceRoute Kitchen**. It takes inspiration from typical food delivery websites like Swiggy/Zomato-style layouts, but uses original text, card design, and styling.

## Tech Used

- HTML
- CSS
- JavaScript
- Bootstrap 5 CDN

## How to Run

Open `index.html` in your browser.
